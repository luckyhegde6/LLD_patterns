# API Gateway — Interview Guide

## 30s Pitch
“The API Gateway is a stateless entry point that applies policies like auth and rate limiting before routing requests to backend services.”

---

## Interruptions

Interviewer: Why gateway?
You: Centralizes cross-cutting concerns.

Interviewer: SPOF?
You: Stateless → horizontally scalable.

Interviewer: Latency?
You: Acceptable trade-off for control.

Interviewer: Where rate limiting?
You: At gateway to protect backends.

Interviewer: Hardest part?
You: Balancing flexibility vs complexity.
